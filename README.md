# pl5736vim
安装说明以ubuntu为例，其余系统请自行斟酌。<br>
##  要求：
为了保证安装成功且正常使用，你需要vim和git。<br>
　　安装vim：<br>
　　　　```sudo apt-get install vim```<br>
　　安装git：<br>
　　　　```sudo apt-get install git```<br>
### 安装前请自行备份现有的vim配置！
## 简易安装：
　　1.获取pl5736vim：<br>
　　　　```git clone https://github.com/pl5736/pl5736vim.git```<br>
　　2.配置pl5736vim：<br>
　　　　```bash ~/pl5736vim/pl5736vim.sh```<br>
