# pl5736vim
安装说明以ubuntu为例，其余系统请自行斟酌。<br>
## 简易安装：<br>
　　1.安装vim和git：<br>
　　　　```sudo apt-get install vim```
　　　　```sudo apt-get install git```
　　2.设置pl5736vim：<br>
　　　　```git clone https://github.com/pl5736/pl5736vim.git```
　　　　```bash ~/pl5736vim/pl5736vim.sh```
　　3.进入vim，输入命令：<br>
　　　　```:PluginInstall```
　　4.退出vim，运行脚本编译YouCompleteMe：<br>
　　　　```bash ~/pl5736vim/pl.sh```
